## The Setup

- Operating System
: ['I use Arch, btw'](https://www.archlinux.org/)
- Display Server
: xorg <sup>[Arch](https://www.archlinux.org/packages/?name=xorg-server)</sup>
- Window Manager
: i3-gaps <sup>[Arch](https://www.archlinux.org/packages/community/x86_64/i3-gaps/)</sup>
- Compositor
: picom <sup>[Arch](https://www.archlinux.org/packages/community/x86_64/picom/)</sup>
- Shell
: fish <sup>[Arch](https://www.archlinux.org/packages/community/x86_64/fish/)</sup>
- Terminal
: termite <sup>[Arch](https://www.archlinux.org/packages/community/x86_64/termite/)</sup>
- Image Viewer
: feh <sup>[Arch](https://www.archlinux.org/packages/extra/x86_64/feh/)</sup>
- Document Viewer
: zathura <sup>[Arch](https://www.archlinux.org/packages/?name=zathura)</sup>
- File Manager
: ranger <sup>[Arch](https://www.archlinux.org/packages/community/any/ranger/)</sup>
- App Launcher
: rofi <sup>[Arch](https://www.archlinux.org/packages/community/x86_64/rofi/)</sup>
- Editor
: code <sup>[Arch](https://www.archlinux.org/packages/community/x86_64/code/)</sup>,
 gvim <sup>[Arch](https://www.archlinux.org/packages/extra/x86_64/gvim/)</sup>
- Web Browser
: google-chrome <sup>[AUR](https://aur.archlinux.org/packages/google-chrome/)</sup>
- Sound Server
: pulseaudio <sup>[Arch](https://www.archlinux.org/packages/community/x86_64/playerctl/)</sup>
- Music App
: spotify <sup>[AUR](https://aur.archlinux.org/packages/spotify/)</sup>
- Screenshot App
: Maim <sup>[Arch](https://www.archlinux.org/packages/community/x86_64/maim/)</sup>
- Font
: San Fransisco Pro <sup>[AUR](https://aur.archlinux.org/packages/otf-san-francisco-pro/)</sup>
- Status Bar
: polybar <sup>[AUR](https://aur.archlinux.org/packages/polybar/)</sup>
- Theming
: python-pywal <sup>[Arch](https://www.archlinux.org/packages/community/any/python-pywal/)</sup>
- Lock Screen
: betterlockscreen <sup>[AUR](https://aur.archlinux.org/packages/betterlockscreen/)</sup>

More:
